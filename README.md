# SENSE YOUR DAY

✨ SENSE YOUR DAY: 당신의 시간을 보듬는 다정한 별지도
"불확실한 오늘 속에서, 나를 온전히 감각하는 시간."

SENSE YOUR DAY는 단순히 미래를 점치는 도구가 아닙니다. 매 순간 마주하는 불확실성이라는 공포를 기술적 통찰과 감성적인 위로로 치환하여, 사용자가 오늘 하루를 온전히 자신의 것으로 느낄 수 있도록 돕는 '감정적 동반자(Emotional Companion)' 프로젝트입니다. 

## Core Features
1. **다정한 문답**: 정해진 운명을 통보하지 않습니다. "오늘 기분은 어때요?"라는 질문으로 당신의 내면을 먼저 들여다봅니다. 
2. **하늘의 조화**: 동양의 사주와 서양의 점성술 데이터를 정밀하게 결합하여, 당신만을 위한 입체적인 에너지 지도를 그립니다. 
3. **운명의 각인**: 당신의 고유한 탄생 정보를 **SBT(Soulbound Token)**로 발행하여, 누구도 침범할 수 없는 디지털 데이터 주권을 실현합니다. 

## Technology Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Database**: Supabase
- **AI**: Gemini API
- **Blockchain**: SBT (Concept)
